device log

Usage Syntax
General $ appbuilder device log --device <Device ID>

Opens the device log stream for a selected connected device.

<Device ID> is the device index or identifier as listed by $ appbuilder device Options:

  • --device - If multiple devices are connected, sets the device for which you want to stream the log in the console.

Related Commands

Command Description

device ios Lists all recognized connected iOS devices with serial number and index.
